## Props

#### AutocompleteInput

| Prop         | Type      | Description                                              | Default |
| ------------ | --------- | -------------------------------------------------------- | ------- |
| showClear?   | boolean   | If true, shows a clear button to reset the input         | false   |
| showTrigger? | boolean   | If true, shows a trigger button to open the dropdown     | false   |
| startAddon?  | ReactNode | Provide startAddon like search icon to show on the front | null    |

#### AutocompletePopup

| Prop         | Type    | Description                                | Default |
| ------------ | ------- | ------------------------------------------ | ------- |
| sideOffset?  | number  | The distance in pixels from the input      | 4       |
| hasBackdrop? | boolean | If true, shows a backdrop behind the popup | false   |
