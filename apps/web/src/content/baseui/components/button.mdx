## Props

#### Button

| Prop       | Type                                                                          | Description                                              | Default   |
| ---------- | ----------------------------------------------------------------------------- | -------------------------------------------------------- | --------- |
| variant    | "default" \| "secondary" \| "destructive" \| "outline" \| "ghost" \| "link"   | The visual style variant of the button                   | "default" |
| size       | "xs" \| "sm" \| "default" \| "lg" \| "xl" \| "icon" \| "icon-sm" \| "icon-lg" | The size of the button                                   | "default" |
| loading    | boolean                                                                       | If true, shows a loading spinner and disables the button | false     |
| disabled   | boolean                                                                       | If true, disables the button                             | false     |
| className? | string                                                                        | Additional class names to style the Button               | \_        |
| render?    | React.ReactElement                                                            | Custom element to render (e.g., Link component)          | \_        |
| children   | ReactNode                                                                     | The content to be rendered inside the Button             | \_        |
